# Projeto Credit Score - EBAC

## 🎯 Objetivo
Este projeto, desenvolvido como parte do curso de Data Science da EBAC, tem como objetivo criar um modelo preditivo de *Credit Score* para classificar o risco de clientes com base em seus perfis financeiros e comportamentais. A meta é automatizar a análise de crédito, permitindo identificar padrões probabilísticos que ajudem na tomada de decisão de forma eficiente.

## 🛠️ Tecnologias Utilizadas
- Python
- Pandas, NumPy
- Scikit-learn (Naive Bayes)
- Matplotlib, Seaborn

## 📊 Metodologia
1. **Exploração dos dados**: Análise inicial do perfil financeiro e tratamento de inconsistências.
2. **Pré-processamento**: Tratamento de variáveis categóricas, balanceamento de classes e separação das bases de treino e teste.
3. **Modelagem**: Aplicação do algoritmo **Naive Bayes** para a classificação do score de crédito.
4. **Avaliação**: Análise da performance do modelo através de métricas de acurácia e capacidade de generalização.
5. **Resultado**: O modelo demonstrou alta eficácia na predição de risco, permitindo identificar precocemente perfis propensos à inadimplência e oportunidades de oferecer melhores taxas a clientes de baixo risco.
